# Brand Guide v1.0 — FMD Soluções Inteligentes

## Logotipo
- Versão Principal: Colorido sobre fundo escuro (website, GitHub, LinkedIn)
- Versão Monocromática Branca: Fundos escuros (contraste)
- Versão Monocromática Preta: Fundos claros
- Versão Horizontal: Cabeçalhos, banners, website
- Versão Quadrada: Avatar GitHub, redes sociais, favicon

## Área de Proteção
Espaçamento mínimo = altura da letra "F"

## Tamanho Mínimo
- Digital: 120px de largura
- Impressão: 25mm de largura
