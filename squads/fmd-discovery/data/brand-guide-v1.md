# FMD Soluções Inteligentes — Brand Guide v1.0

## Paleta de Cores

| Cor | HEX | RGB | Uso |
|-----|-----|-----|-----|
| Azul Primário | `#0A5CFF` | 10, 92, 255 | Botões, links, elementos interativos, ícones |
| Azul Secundário | `#00C2FF` | 0, 194, 255 | Gradientes, destaques, linhas, elementos gráficos |
| Branco | `#F5F7FA` | 245, 247, 250 | Textos principais, logo, ícones |
| Grafite | `#1A1F2B` | 26, 31, 43 | Textos secundários, linhas, elementos discretos |
| Fundo Principal | `#050816` | 5, 8, 22 | Website, dashboards, apresentações |

## Gradiente Institucional
- Início: `#0A5CFF` → Fim: `#00C2FF`
- Uso: Hero, botões, elementos gráficos

## Tipografia

| Uso | Fonte |
|-----|-------|
| Primária (website, interface) | **Geist** |
| Secundária (textos longos) | **Inter** |
| Código (trechos técnicos) | **JetBrains Mono** |

## Estilo Visual
- Minimalista, fundo escuro (`#050816`), alto contraste
- Muito espaço em branco, linhas finas, elementos geométricos
- Visual premium, interfaces limpas

## Elementos Permitidos
Fluxos de dados, arquitetura, circuitos, grid, conexões, geometria, wireframes, estruturas, luz, partículas discretas

## Elementos Proibidos
Robôs, androides, cérebros brilhando, hologramas, cliparts, neon exagerado, ícones caricatos de IA, banco de imagens genérico

## Assinatura Institucional
**FMD Soluções Inteligentes** — *Dados • Automação • Inteligência*
